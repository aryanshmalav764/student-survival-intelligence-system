# Student Survival Intelligence System

Developed by Aryansh Malav

A data-driven platform to help students choose universities in Australia based on budget, field, and lifestyle.

## Features
- Smart filtering (state, city, field, level)
- Budget-based recommendations
- Explainable scoring system
- University comparison tool
- Dynamic images
- Wikipedia-based insights

## Run

pip install -r requirements.txt  
streamlit run app.py
