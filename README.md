# Australia Student Dashboard - Premium Version

**Developed by Aryansh Malav** — an interactive platform designed to explore Australian cities, universities, budgets, and maps dynamically, combining human insight and AI assistance.

## Features

- **City & University Selector:** Explore 20+ cities and 100+ universities across all Australian states, including Tasmania.  
- **Dynamic Backgrounds:** Live city skyline images from Unsplash with Urban/Nature selection.  
- **Budget Simulator:** Compare your monthly budget against rent and living costs for each city.  
- **Interactive Map:** Visualize city and university locations.  
- **Wikipedia Integration:** Auto summaries for selected cities and universities.  
- **Suggestions Box:** Users can submit feedback and ideas.  
- **Live Data Updates:** Universities and images are automatically updated.

## Tech Stack

- Python 3  
- Streamlit  
- Pandas  
- Requests  
- Folium (Interactive maps)  
- Unsplash API  
- Wikipedia API
